Project Alpha: The aim of this F# WebSharper SPA application is that to create a random initial set of matches of competitors by category from a preformatted UTF-8 coded CSV file. The genereted list can be used in a tournament organized as a single-elimination system.

This small application is presumably will be a part of a bigger project with more fucntions.
